
# Slipper

Development tool for @Shopify packing slips.



```sh
deno install                           \
    --force                            \
    --global                           \
    --name slipper                     \
    --allow-write                      \
    --allow-read                       \
    --lock=jsr:@doom/slipper/lock           \
    --import-map=jsr:@doom/slipper/imports  \
    jsr:@doom/slipper
```
