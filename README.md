
# Slipper

Development tool for @Shopify packing slips.



```sh
deno install                           \
    --force                            \
    --global                           \
    --name slipper                     \
    --allow-write                      \
    --allow-read                       \
    --lock=https://jsr.io/@doom/slipper/1.1.0/deno.lock             \
    --import-map=https://jsr.io/@doom/slipper/1.1.0/Imports.json    \
    https://jsr.io/@doom/slipper@1.1.0
```
